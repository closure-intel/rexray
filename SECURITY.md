# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this repository, please report it to us responsibly.

**Please do not open a public issue.**

Instead, email **security@closure-intel.com** with:

- A description of the vulnerability
- Steps to reproduce
- The potential impact
- Any suggested remediation (if known)

We will acknowledge your report as soon as possible and work with you to understand and address the issue promptly.

## Scope

This policy applies to the latest released version of this repository.

Thank you for helping keep Closure and our users safe.
